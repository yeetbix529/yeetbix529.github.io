Tools

recon-tools

A set of modular Bash scripts to automate common bug bounty recon tasks. This is a learning-driven project, and more features will be added over time.

Features

  • Modular Bash workflow
  • Passive and active subdomain enumeration
  • Live host detection with HTTP status code filtering
  • JavaScript scraping and secret detection (multiple versions)
  • Tech stack fingerprinting
  • Wrapper system to chain all tools (WIP)
